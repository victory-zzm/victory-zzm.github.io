1.Mongodb数据库连接

1.1简单了解数据库

  1. MongoDB可视化操作软件，是使用图形界面操作数据库的一种方式；

  2. 在一个数据库软件中可以包含多个数据仓库，在每个数据仓库中可以包含多个数据集合，每个数据集合中可以包含多条文档（具体的数据）

  3. 术语 解释说明
    database 数据库，mongoDB数据库软件中可以建立多个数据库
    collection 集合，一组数据的集合，可以理解为JavaScript中的数组
    document 文档，一条具体的数据，可以理解为JavaScript中的对象
    field 字段，文档中的属性名称，可以理解为JavaScript中的对象属性

1.2启动和连接数据库

  1. Mongoose第三方包：l使用Node.js操作MongoDB数据库需要依赖Node.js第三方包mongoose

  2. 在需要用到的目录或文件夹中：使用npm install mongoose命令下载

  3. 在命令行工具中运行net start mongoDB即可启动MongoDB，否则MongoDB将无法连接

  4. 使用mongoose提供的connect方法即可连接数据库

    注意：在创建数据库，当数据库不存在时，Mongodb会自动创建
